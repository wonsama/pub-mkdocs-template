# Admonitions(훈계)

콜아웃이라고도 하는 Admonitions(훈계)는 문서 흐름을 크게 방해하지 않으면서 사이드 콘텐츠를 포함할 수 있는 탁월한 선택입니다

## 설정 (mkdocs.yml)

```yaml
markdown_extensions:
  - admonition
  - pymdownx.details
  - pymdownx.superfences
```

## 사용법

블록은 !!!로 시작하고 그 뒤에 유형 한정자로 사용되는 단일 키워드가 이어집니다. 블록의 콘텐츠는 다음 줄에 네 칸 들여쓰기로 이어집니다:

### 사용예시

```txt
!!! note

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla et euismod
    nulla. Curabitur feugiat, tortor non consequat finibus, justo purus auctor
    massa, nec semper lorem quam in massa.
```

!!! note

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla et euismod
    nulla. Curabitur feugiat, tortor non consequat finibus, justo purus auctor
    massa, nec semper lorem quam in massa.
